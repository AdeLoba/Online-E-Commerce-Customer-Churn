# Online-E-Commerce-Customer-Churn
This project analyzes customer transaction data for a UK-based online retailer to develop actionable customer segmentation and churn prediction models. Using two years of transaction data (Dec 2009 - Dec 2011), the analysis provides data-driven strategies to optimize marketing spend and improve customer retention.

Business Challenge: Marketing team spending 40% of budget on retention with declining ROI. Need to identify high-value customers, predict churn, and implement targeted interventions.
